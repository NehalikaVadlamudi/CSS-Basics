Style Rule

These are the rule that you have to remember while writing styles for your web pages(html documents)

        /* here we call "p" as a selector */
        /* here we call "color" as a property */
        /* here we call "red" as a value to "color" property */
        /* here we call "color:red" as a declaration */
        /* here we call "p{..}" as a rule */
        p{
            color:red;
            background-color:white;
        }
        /* here we call px as a unit */
        p{
            font-size: 20px;
        }
